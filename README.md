This project is intended to create a matching platform between donors and non-profit organizations. 

Stack:
  - React/Redux
  - SASS
  - React-Router v4
  - NodeJS server with Express
  - MongoDB + Mongoose
  - Webpack + Babel


To start the project:
  - run "npm install"
  - run "mongod" - Install MongoDB first -
  - run "npm start"
  - run "npm run build:watch"
